# ภาษาไทยและ Unicode 👋

ข้อความทดสอบ: สวัสดีค่ะคุณโอ — การพิมพ์ภาษาไทยต้องไม่ตัดสระ วรรณยุกต์ หรือ emoji ค่ะ

- ชื่อไฟล์มีช่องว่าง
- path เป็น UTF-8
- reader และ search ต้องแสดง `กรุงเทพมหานคร` ได้ตรงกัน
